# SETR_Proj2

##  How to execute:

```bash
    cd src
    git clone https://github.com/ThrowTheSwitch/Unity
     
    cd build
    cmake ..
    make

    ./main
    ./tests

    cd $PROJECT_ROOT
    $ doxygen Doxyfile
    $ firefox docs/html/index.html
```

##  How generate the documentation:

```bash
    cd $PROJECT_ROOT

    $ firefox docs/html/index.html
```
